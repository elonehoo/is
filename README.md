  <em>Type check values</em>
</p>

## install

```bash
# npm
npm i @elonehoo/is
# yarn
yarn add @elonehoo/is
# pnpm
pnpm i @elonehoo/is
```

## usage

```typescript
import is from '@elonehoo/is'

is.string('elonehoo') //=> true

is('elonehoo') //=> 'string'
```
